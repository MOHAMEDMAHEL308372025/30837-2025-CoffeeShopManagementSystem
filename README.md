# ☕ Coffee Shop Management System

## 📌 Project Overview

The Coffee Shop Management System is a database application developed as part of the **Database Programming** course. The project was designed to automate and simplify the daily operations of a coffee shop using Oracle Database technologies.

The system manages customers, employees, products, suppliers, inventory, orders, payments, and sales reports. It also provides a management dashboard to help monitor business performance and support decision-making.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Manage customer information.
- Manage employee records.
- Organize product categories.
- Store supplier information.
- Manage coffee shop inventory.
- Process customer orders.
- Record customer payments.
- Generate sales reports.
- Monitor inventory levels.
- Improve business decision-making through reports and dashboards.

---

# 🔗 Database Relationships

- One Category can contain many Products.
- One Supplier can supply many Products.
- One Customer can place many Orders.
- One Employee can manage many Orders.
- One Order contains many Order Details.
- One Product can appear in many Order Details.
- One Product has one Inventory record.
- One Order has one Payment.

# ▶ How to Run the Project

1. Install Oracle Database.
2. Open Oracle SQL Developer.
3. Create the database user.
4. Execute all SQL scripts.
5. Insert sample data.
6. Execute PL/SQL scripts.
7. Import the Oracle APEX application.
8. Run the application.
